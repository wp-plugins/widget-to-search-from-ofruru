Add a Custom Search Site with no setup. Just drag and drop a widget and you're done.

== Description ==

Enables a Custom Search Engine for your site via a widget.

### Features:

*	Works with any WordPress Theme
*	Just add a "Widget to search from ofru.ru" on your site.
*	Support the Mode: Multisite
*	Does not replace the default WordPress search. It runs along with it. Use both if you want.

== Installation ==

Installation

1. Click **Plugins** > **Add New** > search for **widget to search from ofru.ru**
1. Activate the plugin through the **Plugins** menu in WordPress
1. Click **Appearance** > **Widgets** > Drag the **Widget to search from ofru.ru** widget to a widget area.
